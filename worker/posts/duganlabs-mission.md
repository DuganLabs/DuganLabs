## Building for Developers, by Developers

DuganLabs exists to prove that modern web development doesn't require endless dependency chains or opaque frameworks. We believe in **transparent, composable, zero-dependency tooling** that developers can understand, audit, and modify.

### Our Core Values

**Simplicity** — Every package does one thing well. We avoid bloat and over-engineering.

**Transparency** — Open source by default. You can read and understand every line of code you depend on.

**Performance** — Lightweight packages, minimal overhead, and edge-ready execution. Measure twice, optimize once.

**Developer Experience** — Great tooling, clear documentation, and an active community. Building should be a joy, not a chore.

### What We're Building

- **BaseNative** — A modular, zero-dependency framework for web applications
- **GreenPut** — Tools for sustainable computing and energy-efficient deployments
- **The Ecosystem** — A marketplace and registry of composable, high-quality packages

### Join Us

We're inviting developers to contribute to BaseNative and build the next generation of composable web tools. Whether you're interested in signal-based reactivity, server rendering, edge computing, or real-time APIs, there's a place for you.

Visit [github.com/DuganLabs](https://github.com/DuganLabs) to get involved.
